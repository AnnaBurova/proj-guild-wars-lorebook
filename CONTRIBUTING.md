# Contributing / Feedback

Thanks for wanting to help.

I do not accept pull requests or translation forks for this project.

If you want to suggest improvements:
- Open an Issue (preferred)
- Or leave suggestions as comments on specific lines/files (GitHub file comments)

Please include:
- A link to the page/file
- The exact line/fragment
- Your suggested correction and (if possible) the original in-game source

Do not submit:
- Full raw dumps of game scripts
- Ripped assets (music, models, textures)
