## 📖 Overview

**Guild Wars Lorebook** is a non-commercial fan lore archive for Guild Wars (GW1 / GW2 and future entries in the series).
It collects dialogue excerpts (in short fragments), notes, structured lore breakdowns, Russian translations, and source references.

**Guild Wars Lorebook** это некоммерческий фан‑архив лора Guild Wars (GW1 / GW2 и будущие материалы серии).
Здесь собираются диалоги (короткими фрагментами), заметки, структурирование, переводы на русский и ссылки на источники.

The project is designed for comfortable reading in Markdown and publishing via GitHub Pages.

Проект создан для удобного чтения в Markdown и публикации через GitHub Pages.

## 🧩 Contents

- My Russian translations.
- Lore notes and commentary.
- Dialogue excerpts with context.
- Images / screenshots (only when necessary and with respect to rightsholder rights).
- Source references (missions / quests / NPCs / locations / links).

- Мои переводы на русский.
- Заметки по лору и комментарии.
- Диалоги и реплики с контекстом.
- Картинки и скриншоты (только по необходимости и с учётом прав правообладателей).
- Ссылки на оригинал (миссии / квесты / NPC / локации / ссылки).

## 🧠 Contribution policy

[CONTRIBUTING](CONTRIBUTING.md) — Style and contribution rules

Пулл‑реквесты не принимаются.

Если хочется что-то исправить или добавить:
- Создай Issue (предпочтительно).
- Или оставь комментарий к строкам/файлам прямо в репозитории (GitHub file comments).

Что особенно полезно в предложениях:
- Ссылка на конкретный файл/раздел.
- Оригинальная реплика/контекст (где в игре встречается: квест/миссия/NPC).
- Предлагаемая правка перевода и объяснение почему.
